⚠️ คำเตือน: นี่คือ HTML preview เท่านั้น คุณต้อง:
  1. ใช้เครื่องมือ/เว็บไซต์สร้าง PNG icons จริง
  2. บันทึกเป็น icon-192x192.png และ icon-512x512.png
  3. วางไฟล์ไว้ในโฟลเดอร์ public/

วิธีสร้าง Icons:

  1. ใช้ฟรีทูลิตี:
    • RealFaviconGenerator.net
    • Favicon Generator
  2. หรือใช้ Design Tools:
    • Figma: Export เป็น PNG 192x192 และ 512x512
    • Canva: สร้าง rounded square icons
  3. Spec ที่ต้องการ:
    • 192x192px PNG (maskable)
    • 512x512px PNG (maskable)
    • พื้นหลัง: gradient #07b53b → #069a31
    • Icon: 📅 หรือ ตัวอักษร

แนะนำ Icon Design:

  • ใช้ gradient สีเขียว (#07b53b → #069a31) ตาม brand colors
  • ใช้ rounded corners (40px radius สำหรับ 192px)
  • Icon ควรมีคอนทราสต์สูง (light color on dark bg)
  • เว้น text หรือ details ที่จะ illegible เมื่อย่อขนาด

Tools ที่แนะนำ:

  • Canva: สร้าง icon ด้วย templates → Export PNG
  • Figma: Design → Export PNG 2x
  • Sketch: Design → Export
  • Online: Canva Cover Maker → resize to 192x192
